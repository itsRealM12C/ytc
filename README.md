# Hotkeys


`F2` - << Backwards
`F3` - >> Fast forward
`F4` - Left arrow like key
`F5` - Right arrow like key
`F6` - OK/enter like key
`F11` - Enters in full screen mode.
`F12` - Open DevTools
`Ctrl + Shift + I` - Open DevTools
`Esc` - Back button
`⌫`/`Backspace` - Back button
`Ctrl + R` - Reloads the page
`S` - Brings you to the search bar
`Ü`/`-` - Voice search
